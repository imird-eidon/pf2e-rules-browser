# Images

Screenshots referenced from the `media` field in `module.json` (used by the
foundryvtt.com package listing gallery) live here.

Current files:
- `menu.png`
- `module1.png`
- `module2.png`

Reference them with the **raw** GitHub URL, not the `blob/` page URL:
```
https://raw.githubusercontent.com/imird-eidon/pf2e-rules-browser/main/images/menu.png
```

Do **not** use `private-user-images.githubusercontent.com` links (the ones
GitHub generates when you paste/drag an image into a text box) — those are
signed and expire after a few minutes.
